# Frontend Battler

A responsive blog card website developed as part of an IIT hackathon project. The goal was to design visually appealing and mobile-friendly blog cards while maintaining a clean user experience across different screen sizes.

## Project Background

This project was created during an IIT-organized hackathon event, where the focus was on frontend design, responsiveness, and user interface development.

## Features

* Responsive blog card layout
* Mobile-friendly design
* Modern card-based UI
* Image integration
* Clean and organized structure
* Cross-device compatibility

## Tech Stack

* HTML5
* CSS3
